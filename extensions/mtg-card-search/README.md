# MtG Search

Search for Magic the Gathering cards using Scryfall API.

- All card faces illustrations
- Metadata quickly display
  - Game changer indicator
  - Set
  - Mana cost
  - Rarity
  - Keywords
  - Avg. Price
  - Artist
- Follow links to:
  - CardMarket
  - TcgPlayer
  - CardHoarder
  - EDHRec
  - Scryfall

> ℹ️ No additional configuration is required
