# Dub Link Shortener

Dub.co is the open-source link management infrastructure. This extension allows you to shorten and retrieve your Dub links directly inside Raycast.

# Instructions

You need 3 things from Dub: 
1) API Key
2) Project Slug
3) Project Domain
