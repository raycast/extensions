# Phind Search

A Raycast plugin to search anything with Phind.

Inspired by [google-search](https://github.com/raycast/extensions/tree/05c03625c58bda030eaa7f9be2138afe59f96492/extensions/google-search)

## Feature
- Auto suggest
- With context
- Search history
