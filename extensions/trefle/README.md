# Trefle

Lookup information about plants using the [Trefle API](https://trefle.io/).

## Authentication

- Create an account on [trefle.io](https://trefle.io/)
- Confirm your email address
- Login on your account
- Copy the Trefle API token from your [account settings page](https://trefle.io/profile) and paste it into your settings

## Supported Commands

- Search plants
  - Copy scientific name
  - Copy information (JSON format)
  - Show images
