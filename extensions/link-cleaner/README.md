# Link Cleaner

Quickly remove tracking parameters in URLs from clipboard.

Supported:
1. Google Search
2. Baidu Search
3. Bing Search
4. Netease Music
5. Youtube
6. Instagram

## Example

```
"Here is a link to Google: https://www.google.com/search?q=link%2Bcleaner&oq=link%2Bcleaner&aqs=chrome..69i57xxxx&sourceid=chrome&ie=UTF-8"
```
