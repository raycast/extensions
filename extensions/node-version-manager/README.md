## Getting Started

Starting the Development Environment

### Prerequisites


- NVM
  ```sh
    brew install nvm
  ```

> [!NOTE]  
> For NVM you'll need to have a `~/.zshrc` loading nvm to use it

OR 


- FNM
  ```sh
    brew install fnm
  ```
