# Simple flomo Changelog

## [Initial Version] - 2023-09-15

- 1.0 GPT-4 随意发挥
- 笔记内容输入
- 可通过 cmd + shift + enter 预设至多 9 个标签 / 标签组，留白即不显示
- 可自行输入文字添加其他标签
- 可根据偏好把标签置于笔记前 / 后
- 可添加参考文献链接，自动处理为 [1] + 链接的格式，例如：
  - 参考文献
  - [1] google.com
  - [2] baidu.com

## [Bug fix] - 2023-09-15

- API 输入错误报错提示
- tag 在后的情况下应置于参考资料之后
- 修复了多余的换行符问题
