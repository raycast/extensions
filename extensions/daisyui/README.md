## Installation

1. Open Raycast
2. Go to Extensions
3. Search for "daisyUI"
4. Install the extension

## Usage

### Colors Command

- Browse daisyUI color palette
- Filter colors by category
- Copy:
  - Color names
  - Hex codes
  - OKLCH values
  - CSS variables

### Components Command

- Explore daisyUI components
- View descriptions and examples
- Copy:
  - HTML snippets
  - JSX code
  - Component URLs

## Compatibility

- Raycast: Latest version
- daisyUI: v5 (beta 6)
