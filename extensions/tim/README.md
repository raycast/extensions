  <h1 align="center">Tim</h1>
</p>

A [Raycast](https://raycast.com/) extension that lets you manage [Tim](https://tim.neat.software/). Tim is a simple time tracker for macOS.

**Commands:**

- Toggle current active time record
- Create new task
- Open active record
- Open navigator
- Open task manager
- Export time records
- Import time records
