Raycast Note Searcher

A powerful Raycast extension for semantic note searching and organization, powered by Upstash Vector Database.

This is the documentation site for the Raycast Note Searcher extension.

Features

Add Notes

Quickly add notes with automatic categorization based on content.

Add Reminders

Create reminders with date and time selection for follow-ups.

Add URLs

Add web content with automatic metadata extraction for easy reference.

Use Templates

Create notes from predefined templates for consistent formatting.

Search Notes

Search your notes semantically to find what you need quickly.

Installation

This extension is designed for Raycast. To install from source:

  1. Clone the GitHub repository
  2. Navigate to the project directory
  3. Install dependencies: npm install
  4. Build and install the extension: npm run dev
