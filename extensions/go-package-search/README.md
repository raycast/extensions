# Go Package Search

This [Raycast](https://www.raycast.com) extension allows you to search Golang packages on [pkg.go.dev](https://pkg.go.dev).

> Warning: This extension uses go.dev to supply its data, and by using this extension you consent to be bound by [Go's Terms of Service](https://go.dev/tos). Use at your own risk.

Assets taken from the [Go Brand Guidelines](https://go.dev/blog/go-brand).
