# Toncoin Price

Keep an eye on Toncoin (TON) cryptocurrency price from your system menu bar. Stay informed and up-to-date with real-time price updates at a glance.

## Features

**Supported Sources**

- OKX (default)
- Bybit

**Useful Links**

Display useful links in menu bar:

- CEX spots
- TON blockchain DEXs
- CoinMarketCap and CoinGecko
