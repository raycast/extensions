# Toggle Menu Bar

Switch between show/hide menu bar modes.

## Troubleshooting

```diff
+ Green Toast    → menu bar is shown
- Red Toast      → menu bar is not shown
```

### Error: Command failed with exit code 1: osascript -e

`execution error: Not authorised to send Apple events to System Events.`

In order to use this extension, you must enable the Automation permission for Raycast to control System Events.
