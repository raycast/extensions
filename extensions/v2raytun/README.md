# V2raytun Raycast Extension

This Raycast extension allows you to quickly connect to your VPN using v2raytun via a URL scheme.

## Features

- **Quick Connection:**  
  Launch the command to initiate the VPN connection.
- **Raycast Integration:**  
  Easily access the extension through the Raycast command palette.
- **Simple Configuration:**  
  Minimal setup is required to tailor the extension to your needs.
