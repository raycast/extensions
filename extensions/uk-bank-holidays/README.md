# UK Bank Holidays Raycast Extension

This Raycast extension provides upcoming bank holiday information for England and Wales.

## Features

- List of all upcoming bank holidays.
- Information about the next bank holiday.

## Usage

You can activate the extension by navigating to the "Extensions" tab on Raycast and searching for "UK Bank Holidays".

The extension provides two commands:

- **Show Future Holidays**: This command displays a list of all future bank holidays in England and Wales. Each list item shows the date and the official name of the holiday.
- **Next Holiday**: This command displays only the next bank holiday in England and Wales. It shows the official name and the date of the holiday.

The extensions fetch data from the [UK Government's Bank Holidays API](https://www.gov.uk/bank-holidays.json).
