# Number Verification API Raycast Extension

This is a Raycast extension for [Number Verification API](https://apilayer.com/marketplace/number_verification-api) by [APILayer](https://apilayer.com/). With this extension you can perform lookup for a total of 232 countries around the world and verify the validity of phone numbers.

1. **Install extension**: Click the `Install Extension` button in the top right of [this page](https://www.raycast.com/xmok/verify-number) OR via Raycast Store

2. **Enter your API Key**: The first time you use the extension, you'll need to enter the **API Key** in Preferences OR at first prompt:

    a. `Navigate` to API in Marketplace ([link](https://apilayer.com/marketplace/number_verification-api))

    b. `Subscribe` to plan of your choice

    c. `Copy` API Key and `Paste` in Preferences
