# SnippetSurfer

SnippetSurfer is a raycast extension designed to swiftly navigate through snippets of information. You can effortlessly skim through text excerpts, code snippets, or any other brief segments of content.

### Features
- Raycast extension for quick navigation of text excerpts and code snippets.
- Reads snippets from Markdown files for easy organization.
- Supports Metadata YAML for enhanced categorization.
- Automatically copies code snippets to clipboard for streamlined workflow integration.

### Example of markdown
```
	---
	Title: Git Init
	Description: Create empty Git repo in specified directory. Run with noarguments to initialize the current directory as a git repository
	---
	```shell
	git init
	```
```
