# Zipline Changelog

## [Initial Version] - 2025-07-07

- Browse and search your Zipline uploads
- Upload new files to Zipline with drag & drop support  
- View your most recent uploads
- Instantly upload clipboard text with file type selection
- Support for multiple file types (txt, md, json, js, ts, py, html, css, sql, xml)
- Configurable Zipline instance URL and API token
- Customizable page size for upload browsing
