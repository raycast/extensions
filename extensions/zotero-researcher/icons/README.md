# Icons for Zotero Researcher Extension

You need to add the following icon files to this directory:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Recommended Open Source Icons

You can download suitable open-source icons from one of these sources:

1. **Material Icons** - https://material.io/resources/icons/
   - Search for "research," "library," or "article" icons
   - Download and resize to required dimensions

2. **Font Awesome** - https://fontawesome.com/
   - Icons like "book," "scroll," or "search" would be appropriate
   - Make sure to use icons available under free license

3. **The Noun Project** - https://thenounproject.com/
   - Search for "zotero" or "research" icons
   - Many are available under Creative Commons licenses

4. **Simple Icons** - https://simpleicons.org/
   - They have a Zotero icon available (https://simpleicons.org/?q=zotero)

## Suggested Zotero Icon

A good choice would be to use the official Zotero "Z" logo but with a different color scheme to avoid trademark issues. Make sure to check Zotero's trademark policy before using anything too similar to their official branding.

## Creating the Icons

After downloading a suitable base icon:
1. Resize to 16x16, 48x48, and 128x128 pixels
2. Save as PNG files with the correct filenames
3. Place in this directory
