# Tree Generator

A Raycast extension to generate directory tree structures.

## What it does

- 🌳 Generate visual directory trees
- 📏 Control depth and filtering options
- 🙈 Respect `.gitignore` files
- 📋 Copy results to clipboard

## Commands

- **Generate Directory Tree** - Interactive form with customization options
- **Quick Directory Tree** - Fast generation with preset options
- **Create Files from Tree** - Create directory structure and files from a tree representation

## Use Cases

Perfect for AI analysis workflows - when you need to analyze project file structures with AI tools, this extension provides a clean tree representation that can be easily copied and shared with AI assistants for code analysis, refactoring suggestions, or architectural reviews.
