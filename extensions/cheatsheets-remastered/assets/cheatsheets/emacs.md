### Movements

   ^n ^p    # up/down
   ^f ^b    # left/right

   ^v Mv    # up/down page

   ^a ^e    # begin/end of line
   Ma Me    # begin/end of sentence

### Basic

   ^x ^f    # find file
   ^x ^s    # save file

### Command line

   Mx

### Packages

   Mx package-install RET evil RET
