### Antialias
{: .-prime}

```css
* {
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}
```

### Support

 * Firefox 25+ on OSX
 * Webkits (Chrome, Safari, etc)
