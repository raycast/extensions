# Icon Finder Changelog

## [1.2] - 2024-09-02

- Updates icons to the version 10.6.0 of the WordPress Icon library
- Fixes a typo in the project description

## [1.1] - 2024-05-07

- Adds an ID to the SVG object with the name of the icon so that when it’s pasted in Figma, the frame will have the correct name instead of the generic ‘Frame’.
- Improves the rendering of the icons by explicitly setting their dimensions (minWidth, minHeight, width, and height).
- Adds keywords to several icons to improve the search.
