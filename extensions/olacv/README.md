# OlaCV

This is a Raycast extension for [ola.cv](https://www.ola.cv/) - The Official Registry for .CV Domains. With this extension you can view domains, their DNS Zones, contacts and create new contacts.

## ðŸš€ Getting Started

1. **Install extension**: Click the `Install Extension` button in the top right of [this page](https://www.raycast.com/xmok/olacv) OR `install` via Raycast Store

2. **Get your API Key**:

    Please see the Developer Docs at https://docs.ola.cv/ to learn how you can create your developer account and get the API Key.

> TIP: Check out [Testing](https://docs.ola.cv/api/testing.html) to see how you can test before going fully live.
