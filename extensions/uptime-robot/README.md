# UptimeRobot

This is a Raycast extension for [UptimeRobot](https://uptimerobot.com/) - The world's leading uptime monitoring service.

## üöÄ Getting Started

1. **Install extension**: Click the `Install Extension` button in the top right of [this page](https://www.raycast.com/xmok/uptime-robot) OR `install` via Raycast Store

2. **Enter your UptimeRobot API Key**:

    - `Follow` instructions from https://uptimerobot.com/api/#auth to get the Key
    - `Copy` and `Paste` this in Preferences
