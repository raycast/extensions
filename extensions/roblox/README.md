## Features
- Searching for Games
- View Games with ID
- View User Info With ID or Username
- View Your Own User Profile
- Launches Roblox Games in Roblox Client & Studio
- Favourite Games and find them easily
- View a Favourite Game's Player Count on the Menu Bar
