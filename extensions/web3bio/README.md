# Web3.bio for Raycast

[Web3.bio](https://web3.bio) is a platform for Web3 and Web 2.0 Identity Graph search and link in bio profiles. It provides a list of relevant identities when searching for a Twitter handle, Ethereum address, ENS, Basenames, Linea domain, Lens profile, Farcaster account, and other Web3 identities.

This extension utilizes the [Web3.bio Profile API](https://api.web3.bio) to search for Ethereum, ENS, Basenames, Linea domains, Lens, Farcaster, and other Web3 domains and identities.
