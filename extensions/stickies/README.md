# Stickies

Search and show macOS's built-in Stickies via Raycast.

## Search Stickies

- Support content search
- Support Markdown format
- Support TXT format
- Sort by update time
- Save as Snippet

## Menubar Stickies

- Support TXT format
- Sort by update time
- Show recently updated Stickies in the menu bar
- Click on the note to copy or paste it to active app
