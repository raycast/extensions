# Unleash Feature Toggle
![unleash-feature-flag-3](https://github.com/firhanram/unleash-raycast-extension/assets/59257112/e821370c-53a0-4ba5-8f9e-29152e6d9c08)

This is the [Raycast](https://www.raycast.com/) extension for toggling feature flags that registered in [Unleash](https://www.getunleash.io/). It helps you quickly toggle a feature from anywhere on your mac. It's packed with features like,
   1. Create feature toggle
   2. Enable or disable feature toggle
   3. Archive feature toggle

# Setting up the Extension
To use this extension you must add your Unleash API URL `ex: https://<your-unleash-instance>/api` and [Personal Access Token](https://docs.getunleash.io/reference/api-tokens-and-client-keys#personal-access-tokens). You can read the guide how to create the personal access token [here](https://docs.getunleash.io/how-to/how-to-create-personal-access-tokens). Make sure you have permission to create, update and read.
