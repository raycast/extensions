## Raycast Wallpaper

Set Raycast official wallpapers and capture for Raycast extension's metadata.

**Commands**

- **Raycast Wallpaper**
  - Get and set Raycast wallpapers to desktop.
- **Random Wallpaper**
  - Set random Raycast wallpapers to desktop.
