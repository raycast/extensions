# Color Casket

![Preview Screenshot](./metadata/screenshot-1.png)

Handy color utilities in your pocket.

**Color Casket** provide you simple ability to render color strings in next formats:

- HEX
- RGB
- HSL
- Keyword

Keyword - its basic color names as: red, blue, cyan, purple, etc. When you searching in another format **Color Casket** will find closest color name;

## Usage

Use `Render Color` command to run.

Available keywords to find command: `color`, `render`, `render color`, `color utilities`, `color casket`

## Features

- [x] Render Color by String Value
- [x] Copy History
- [x] Favorites Color List
