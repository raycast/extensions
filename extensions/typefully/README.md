# Typefully

The Typefully extension to help you quickly interact with the platform, and reach your audience.

## Requirements

- You will need a Pro account with Typefully in order to generate and use an API key.

## Getting your personal API key.

1. First things first, open [Typefully](https://typefully.com/) and click on your profile icon in the top left hand corner of the screen.
2. Next, select the Integrations menu item.
3. Copy your key under the API Keys section. If you don't have a key yet, generate a key here, then copy it.

You will need this key to use the Typefully Raycast extension.
