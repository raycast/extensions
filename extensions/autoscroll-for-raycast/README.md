# Autoscroll for Raycast

Control automatic scrolling in any direction, right from
[Raycast](https://raycast.com).

## Commands
- [ ] Autoscroll Downward
- [ ] Autoscroll Upward
- [ ] Autoscroll Left
- [ ] Autoscroll Right
- [ ] Stop Autoscroll
