# Word Calc

Calculate word count, reading time, and speaking time for any text with this Raycast extension.

![Word Calc](.github/img/raycast_extension.png)

## ✨ Features

- ⏱️ **Time Estimates** - Calculates reading and speaking duration
- 📈 **Detailed Stats** - Word count, character count, and paragraph count
- ⚙️ **Customizable Speeds** - Adjust reading/speaking speeds to match your pace
- 📋 **Quick Copy** - Copy individual stats or all stats with one click

## 🚀 Usage

1. **Copy text** to your clipboard (from any source)
2. **Launch Word Calc** in Raycast
3. **View instant analysis** with all statistics
