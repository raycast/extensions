# Developer Tools Extension for Raycast

A comprehensive collection of essential developer tools that enhance your productivity right from Raycast. From JSON formatting to JWT debugging, these tools are designed to streamline your development workflow.

## ‚ú® Key Features

- **JSON Tools**: Format and validate JSON with syntax highlighting
- **JWT Debugging**: Decode and inspect JWT tokens with detailed analysis
- **Time Utilities**: Convert between Unix timestamps and human-readable dates
- **String Manipulation**: Generate random strings, UUIDs, and handle Base64 encoding
- **Markdown Support**: Real-time preview and editing of markdown content
- **Security Tools**: Generate secure passwords with customizable options

## üöÄ Available Commands

### JSON Format/Validate
- **Function**: Format and validate JSON input
- **Usage**: Paste JSON, get formatted and validated output with syntax highlighting
- **Features**: 
  - Syntax validation
  - Pretty printing
  - Copy formatted output
  - Error highlighting

### JWT Debugger
- **Function**: Decode and inspect JWT tokens
- **Usage**: Paste JWT token to view decoded header and payload
- **Features**:
  - Header and payload analysis
  - Signature verification status
  - Expiration time display
  - Copy decoded values

### Unix Time Converter
- **Function**: Convert between Unix timestamps and dates
- **Usage**: Enter timestamp or date to convert
- **Features**:
  - Bidirectional conversion
  - Multiple date formats
  - Current time display
  - Copy converted values

### Random String Generator
- **Function**: Generate random strings with custom options
- **Usage**: Configure length, character types, and generate
- **Features**:
  - Customizable length
  - Character type selection
  - Multiple string generation
  - Copy to clipboard

### UUID Generator
- **Function**: Generate UUID v4 and v7
- **Usage**: Select version and generate UUIDs
- **Features**:
  - Support for UUID v4 and v7
  - Bulk generation
  - Copy to clipboard
  - Format options

### Base64 Encode/Decode
- **Function**: Convert text to/from Base64
- **Usage**: Enter text to encode or Base64 to decode
- **Features**:
  - Bidirectional conversion
  - URL-safe encoding
  - Copy encoded/decoded text
  - Error handling

### Markdown Preview
- **Function**: Preview and edit markdown in real-time
- **Usage**: Write markdown with live preview
- **Features**:
  - Real-time preview
  - Syntax highlighting
  - Export options
  - Common markdown shortcuts

### Password Generator
- **Function**: Generate secure passwords
- **Usage**: Configure options and generate passwords
- **Features**:
  - Customizable length
  - Character type selection
  - Password strength indicator
  - Copy to clipboard
