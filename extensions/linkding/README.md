# Preferences

| Properties               | Label                  | Value                               | Required | Default | Description                                                                                                      |
| ------------------------ | ---------------------- | ----------------------------------- | -------- | ------- | ---------------------------------------------------------------------------------------------------------------- |
| `baseURL`                    | linkding url                    | `string`                            | `true`   | `empty` | The base URL of your linkding installation, without the /bookmark path or a trailing slash                                                                                    |
| `token`              | Token      | `string`                           | `true`   | `empty`  | Used to authenticate against the linkding API. You can find this on your linkding settings page
