# Hide Files

Hide files from any directory, including desktop, downloads, texts, etc.

**Command**

- **Hide Desktop Files**: Hide All desktop files.
- **Show Desktop Files**: Show hidden desktop files.
- **Hide Selected Files**: Hide selected files in the current window of the Finder.
- **Hide All Files**: Hide all files in the current window of the Finder.
- **Show All Files**: Show all hidden files in the current window of the Finder.
- **Manage Hidden Files**: Manage all the files you hide with this extension.
  - Open in Finder/Open in Default App
  - Reveal in Finder
  - Show File
  - Show All Files


You can use the system shortcut "⇧+⌘+." to see all system hidden files.
