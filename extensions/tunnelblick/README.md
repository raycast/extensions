# Tunnelblick

## About

This extension interacts with [Tunnelblick](https://tunnelblick.net) to interact
with configurations.

## Commands

### `Tunnelblick`

Uses Apple Script to fetch configurations (opens Tunnelblick in the background).  
Available actions:

- Connect/Disconnect (`↩`)
- Disconnect All (`⌘ + ⌫`)
