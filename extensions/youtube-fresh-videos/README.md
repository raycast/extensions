# YouTube Fresh Videos

Stay up to date with your favorite YouTube channels directly from Raycast.

## Features

- Follow multiple YouTube channels
- View latest videos from all your followed channels
- Filter videos by channel
- Quick access to video links
- Easy channel management

## How to Get YouTube Channel ID

To add a channel to YouTube Fresh Videos, you'll need its Channel ID. Here's how to find it:

1. **Using Channel URL**
   - Go to [YouTube Channel ID Finder](https://timeskip.io/tools/youtube-channel-id-finder)
   - Enter the YouTube channel URL (e.g., `https://youtube.com/@channelname`)
   - The tool will display the channel ID

2. **Manual Method**
   - Go to the YouTube channel's page
   - Click on the "About" tab
   - Right-click anywhere on the page and select "View page source"
   - Press âŒ˜F and search for "channelId"
   - The channel ID will be a 24-character string starting with "UC"

Example Channel ID: `UCRlsJWh1XwmNGxZPFgJ0Zlw`

## Tips

- You can add as many channels as you want
- Videos are sorted by publication date
- Use the search bar to find specific videos
- Filter by channel to focus on content from a specific creator
