# Toggle Fn Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2025-08-17

### Added

- Comprehensive AppleScript implementation supporting macOS 13-15
- State verification before and after toggle operation
- Support for multiple UI element locations for better compatibility
- Minimal system notifications for essential feedback only
- Enhanced error handling with meaningful error messages
- Accessibility permissions guidance in troubleshooting
- Production-ready implementation with clean user experience

### Changed

- Complete rewrite of AppleScript logic for improved reliability
- Updated to work with latest macOS versions (Sequoia 15.x)
- Improved UI automation for System Settings navigation
- Better compatibility across different macOS versions

### Fixed

- Resolved issues with System Settings navigation on newer macOS versions
- Fixed checkbox detection across different UI layouts
- Improved dialog closing mechanism
- Enhanced error recovery when UI elements are not immediately available

## [1.0.0] - 2024-08-25

### Added

- Initial version code based on MrSimonC's Toggle-Mac-Function-Keys project
