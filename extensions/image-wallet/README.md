# Image Wallet

## Folder Structure

The Wallet Directory can have multiple Pockets and Cards within it. Pockets can have multiple Cards, but no sub-Pockets.

- Wallet Directory
  - Folder
    - File
    - File
  - Folder
    - Folder (NOT VISIBLE IN WALLET)
      - Image (NOT VISIBLE IN WALLET)
    - File
    - File
    - File
  - File
  - File
