## [1.0.0] - {PR_MERGE_DATE}

### Added

* Initial release of Gmail - Single Email
* Command "Single Email" to view the latest email in the inbox
* Email details shown: subject, sender, recipients, date, plain text body, and attachments
* Actions: Open in Browser, Download Attachment, Mark Read/Unread, Archive, Delete, Copy Web URL, Copy ID, Refresh
* Requires user-provided Google Cloud OAuth Client ID for authentication
