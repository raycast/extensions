# YouTube Music for Raycast
Search and play music from YouTube's vast library directly through Raycast. Find your favorite songs, albums, playlists, and artists without leaving your keyboard


## Features
- Search across YouTube Music's vast library
- Filter results by songs, albums, playlists, and artists
- More features coming...

## Commands

### Search
Search for music from YouTube's library and select what to play. Use the dropdown menu to filter your search by songs, albums, playlists, or artists
