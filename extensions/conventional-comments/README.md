### Configuration

Quick breakdown of what is available:

- type (dropdown)
- description (manual entry)
- scope (manual entry)
- body (manual entry)
- breaking change (manual entry)

#### Format(s)

Three available:

1. `{emoji}{scope} {subject}`
1. `{emoji}{scope}: {subject}`
1. `{type}{scope}: {subject}`
