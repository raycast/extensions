# Changelog

## [1.0.0] - 2025-06-30

### Added
- **Open with Windsurf** - Quick file/folder opening in Windsurf IDE
- **Windsurf Projects** - View and manage recent projects with timestamps and icons  
- **Add Project** - Manual folder selection with native macOS file picker (Cmd+N)
- Project actions: Show in Finder, copy path, remove from list
- Path expansion support and automatic project saving
