# Vortex
The fastest way to use bitcoin on macOS

Control you bitcoin lightning wallet directly from Raycast

## How to connect your wallet

You can connect any wallet that supports Nostr Wallet Connect (NWC):

- [Alby Accounts on getalby.com](https://nwc.getalby.com/)
- [Mutiny](https://www.mutinywallet.com/)
- [Umbrel](https://apps.umbrel.com/app/alby-nostr-wallet-connect)
- [Start9](https://start9.com/)

To connect, create new app connection and paste generated pairing secret into Vortex

## Features

- Pay invoices
- Create invoices
- View balance
- View transaction list
- See transaction details
