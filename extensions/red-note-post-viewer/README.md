# Red-note Post Viewer

Easy to view Red-note post like in browser

## Usage

1. Set cookie
2. Free to go

## How to get cookie

1. Open RedNote with browser([Link](https://www.xiaohongshu.com/explore))
2. Login
3. Open browser developer tools and copy cookie
