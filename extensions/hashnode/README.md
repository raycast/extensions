## Features

- [x] Search your stories
- [x] Read featured stories
- [x] Read latest stories
- [x] Read best stories
- [x] Open new draft story
- [ ] Create draft story with pre-filled (waiting for Hashnode)
- [ ] Search all your drafts (waiting for Hashnode)

and more to come

## How to get your Hashnode Personal Access Token (PAT)

- Go to [account settings > developer](https://hashnode.com/settings/developer)
- Get or generate a token

## How to get your Hashnode Username

- Go to [Profile](https://hashnode.com/settings)
- Then search for `Profile Identity` you will find your username inside
