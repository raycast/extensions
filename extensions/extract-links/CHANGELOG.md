# 更新日志

此文件记录"提取网页链接"Raycast扩展的所有重要变更。

## [1.0.0] - 2023-05-22

### 新增
- 初始版本发布
- 支持从当前浏览器页面提取所有链接
- 以Markdown格式列表输出链接
- 自动处理相对链接，转换为完整URL
- 自动复制到剪贴板以便快速使用
- 显示操作状态的Toast通知和HUD提示
- 增加错误处理和用户友好的提示信息

### 技术特性
- 使用Raycast浏览器扩展API获取页面内容
- 正则表达式解析HTML提取链接
- 智能处理各类URL格式（绝对路径、相对路径等）
- 提取链接文本作为Markdown链接标题
