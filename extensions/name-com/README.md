# Name.com

This is a Raycast extension for [Name.com](https://name.com/). With this extension, you can view Account Balance, Domains, DNS Records, and Delete DNS Records.

## ðŸš€ Getting Started

1. **Install extension**: Click the `Install Extension` button in the top right of [this page](https://www.raycast.com/xmok/name-com) OR `install` via Raycast Store

2. **Get your API Key**: The first time you use the extension, you'll need to enter your 'Name.com' API Key:

    a. `Follow` [instructions](https://www.name.com/support/articles/360007597874-signing-up-for-api-access)

    b. `Enter` credentials in Preferences OR at first prompt
