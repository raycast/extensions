# Tmux Sessioner
## Description
This is a extension for raycast to manage tmux sessions.

## Features
- Switch between sessions 🔄
- Attach to sessions automatically with setup terminal 🖥
- Create new sessions 🆕
- Delete sessions 🗑
- Rename sessions 📝
