# Universities

Search for universities worldwide by name.

## Setup 
You need an API key for using Geoapify static maps API. 
Get it on https://www.geoapify.com/get-started-with-maps-api

## Credits 

- [University Domains and Names API](https://github.com/Hipo/university-domains-list-api)
- [Geocode](geocode.maps.co) for geolocating 
- [Geoapify](https://www.geoapify.com/) for static maps generation
