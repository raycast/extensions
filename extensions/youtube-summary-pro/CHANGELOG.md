# Changelog

All notable changes to the YouTube Summary Pro extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-23

### Added

- Initial release of YouTube Summary Pro
- AI-powered video summary generation
- Detailed timestamps with section breakdowns
- Search and history functionality
- One-click summary copying
- Collapsible detailed summaries
- Command+Shift+C shortcut for copying
- Support for YouTube video transcripts
- Search functionality in summary history
