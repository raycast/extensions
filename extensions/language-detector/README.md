# Language Detector

Detect languages through Raycast AI or third-party language detectors

[![raycast-cross-extension-badge]][raycast-cross-extension-link]
[![raycast-pro-enhanced-badge]][raycast-pro-enhanced-link]

## Features

- **Type to Detect** - Quick detect the language from your text
- **Text to Text** - Input your text or paragraph to detect
- **Selected Text to Detect** - Detect the selected text on any application

## API

This extension follows [Raycast Cross-Extension Conventions][raycast-cross-extension-link].

See [API.md](https://github.com/raycast/extensions/tree/main/extensions/language-detector/API.md).

## Supported Detectors

- [Raycast AI](https://developers.raycast.com/api-reference/ai) (require [Raycast Pro](https://raycast.com/pro))
- [npm:franc](https://npmjs.com/franc)
- [npm:languagedetect](https://npmjs.com/languagedetect)
- [npm:tinyld](https://npmjs.com/tinyld)
