# What The Time - Raycast Extension

A Raycast extension that shows current time in multiple locations right in your menu bar. Perfect for teams working across different time zones or anyone who needs to keep track of multiple time zones at once.

## Features

- 🕒 Shows current time for multiple time zones in the menu bar
- 🔄 Auto-updates every 30 seconds
- 🌍 Includes common timezone suggestions with region-appropriate emojis
- 🔍 Search and add any timezone
- ✏️ Customize emoji indicators for each timezone
- 🎯 Quick access from menu bar

## Installation

1. Make sure you have [Raycast](https://raycast.com/) installed
2. Install the extension from the [Raycast Store](https://raycast.com/store)

## Usage

1. Open Raycast and search for "What The Time"
2. Use "Launch" to show the time in your menu bar
3. Use "Add / Remove" to manage your time zones
4. Click on the menu bar item to see detailed times

## Development

```bash
# Install dependencies
npm install

# Start development
npm run dev

# Build the extension
npm run build
```
