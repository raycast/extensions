## Features

## How to get the API key

- Goto to https://pixabay.com/api/docs
- Move down to the `Parameter` list of `Search Images` section
- Press on `Login` next to the parameter `key` (if not already logged-in) and click on the link for the key if the API key is not already shown
- You should see now the API key which is required for this extension
