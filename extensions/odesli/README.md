# Odesli

Get an Odesli link for the currently copied link of a song/album. Odesli is a service that provides smart music links. These links show a list of platforms you can hear the song/album on.

## Supported platforms

- spotify
- itunes
- appleMusic
- youtube
- youtubeMusic
- google
- googleStore
- pandora
- deezer
- tidal
- amazonStore
- amazonMusic
- soundcloud
- napster
- yandex
- spinrilla
- audius
- anghami
- boomplay
- audiomack
