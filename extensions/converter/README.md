A [Raycast](https://raycast.com) extension that lets you easily convert various kinds of data.

## Feature

- String
  - Base64 Coding
  - URL Coding
  - Hashing
    - MD5
    - SHA1
    - SHA256
- Number
  - Binary
  - Octal
  - Decimal
  - Hexadecimal
  - Bit Swap
- IP
  - String
  - Number
