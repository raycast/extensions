# BetterDisplay

This extension allows you to control your display using [BetterDisplay](https://betterdisplay.pro) from the comfort of Raycast.

## Features

- Turn display on/off
- Turn PIP on/off
- Change display resolution
- Change display brightness
- Change display contrast
- Interact with your displays using Raycast AI (Raycast Pro required)
