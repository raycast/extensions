# Laravel Herd

Provides quick access to Laravel Herd features and settings

# Development

1. Clone repository
2. `npm install`
3. `npx ray dev`

# Distribution

1. Run `npx ray lint`
2. Run `npx ray build` to create a build in the output directory. Install this build locally and test it.
