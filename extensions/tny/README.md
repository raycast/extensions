# Tny

A [Raycast](https://raycast.com) plugin for creating and managing [Tny](https://tny.app) links.

## Usage

You will need to have a Tny account to use this plugin. You can sign up for free at [tny.app](https://tny.app).

## Features

- Managing your Tny links (list, copy, delete)
- Creating new Tny links from URL on your clipboard
- Replacing the currently selected URL with a Tny link
