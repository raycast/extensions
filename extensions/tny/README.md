# Tny

A [Raycast](https://raycast.com) plugin for creating and managing [Tny](https://tny.app?ref=raycast-tny) links.

Tny is a URL shortener that allows you to create short links for your long URLs. Tny is part of [Chief Tools](https://chief.app?ref=raycast-tny).

## Usage

You will need to have a Tny account to use this plugin. You can sign up for free at [tny.app](https://tny.app?ref=raycast-tny).

## Features

- Managing your Tny links (list, search, copy, delete)
- Creating new Tny links from URL on your clipboard
- Replacing the currently selected URL with a Tny link
